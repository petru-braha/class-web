Acest director contine un exemplu de procesare cu PHP5 a documentelor HTML 
(invalide) pe baza functionalitatilor oferite de biblioteca 
Simple HTML DOM Parser <http://simplehtmldom.sourceforge.net/>.

* scraping.php - program care preia informatii despre produse (albume muzicale)
                 de pe un sit Web de profil
